
# 🛒 Supermarket Sales Analysis 可视化项目

本项目为使用 Python 进行的超市销售数据分析，目标是通过数据可视化快速掌握销售趋势、利润结构与分类表现，适合作为数据分析转职/实习项目展示。

---

## 📊 使用的工具与技术

- **Python**：核心语言
- **Pandas**：数据处理与清洗
- **Matplotlib & Seaborn**：数据可视化
- **Jupyter Notebook**：交互式分析流程
- **GitHub**：项目托管与版本控制

---

## 📁 项目结构说明

| 文件/目录 | 说明 |
|-----------|------|
| `day3_review.md` | Day 3 的可视化分析总结（含代码+解释） |
| `day3_charts/` | 输出图表截图，用于成果展示 |
| `supermarket_sales.csv`（或data文件夹） | 原始或处理后的销售数据 |
| `notebooks/` | 后续分析 notebook 文件保存位置 |
| `README.md` | 项目介绍文件（即本页） |

---

## 📈 分析图表示例

- 销售额分布直方图（含 KDE 密度曲线）
- 每月销售趋势线图
- 各区域销售/利润柱状图
- 各商品子类利润排序柱状图（横向）

📎 可见于：`day3_charts/` 或 notebook 执行输出
